# 회원사 등록/수정 — 엑셀로 관리하기

회원사 목록(메인페이지 "대구·경북 회원사 네트워크" 지도 섹션)을
**엑셀 파일 하나로** 관리합니다.

## 📄 관리 파일
- **`회원사_관리.xlsx`** ← 이 파일만 편집하면 됩니다. (프로젝트 폴더 안)

## ✏️ 회원사 추가/수정 방법
1. **`회원사_관리.xlsx`** 를 엑셀로 엽니다.
2. 표에 회원사 정보를 입력합니다. (한 줄 = 한 회원사, **아래로 계속 추가 가능 — 개수 제한 없음**)
   | 열 | 설명 | 필수 |
   |----|------|:----:|
   | 업체명 | 회원사 이름 | ✅ |
   | 전문분야 | 예: 서버·보안 솔루션 | |
   | 지역 | 목록에 표시되는 칩, 예: 대구 / 경북 포항 | |
   | 주소 | **지도 핀 위치로 사용** — 정확할수록 정확히 표시 | ✅ |
   | 대표자 | | |
   | 대표전화 | | |
   | 홈페이지 | 있으면 버튼으로 표시 | |
   > 첫 행의 **열 제목(헤더)은 바꾸지 마세요.** (열 순서는 바꿔도 됩니다)
3. 저장(덮어쓰기)합니다.
4. 터미널(프로젝트 폴더)에서 아래 한 줄 실행:
   ```
   npm run import:partners
   ```
   → `src/data/partners.ts` 가 자동으로 갱신됩니다.
5. 사이트에 반영(배포):
   ```
   git add -A
   git commit -m "회원사 갱신"
   git push
   ```
   1~2분 뒤 https://hanbyeolsystem.github.io/dk-it/ 에 반영됩니다.

## 🆘 자주 묻는 것
- **엑셀 파일을 실수로 지웠어요** → `npm run make:partners-template` 실행하면
  기본 양식(`회원사_관리.xlsx`)이 다시 만들어집니다. (단, 기존 입력 내용은 사라집니다)
- **"업체명 열을 찾지 못했습니다" 오류** → 첫 행 헤더에 "업체명"이 들어간 칸이 있는지 확인하세요.
- **지도 핀이 엉뚱한 곳** → 주소를 더 구체적으로(도로명+건물번호) 입력하면 정확해집니다.

> 매번 직접 하기 번거로우면, 엑셀만 채워서 저한테 주셔도 됩니다 — 반영·배포까지 대신 해드립니다.
